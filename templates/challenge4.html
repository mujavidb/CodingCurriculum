So to be able to create organised and reusable programs you will need to be able to create functions. A function is a block of organised/reusable code that is used to perform a single, related action. By doing so you will make your program more readable and you will also find by making your programs into smaller chunks of functions your original problem becomes much easier.

So a few rules when you are defining a function:

  • Function blocks begin with the keyword def followed by the name you choose and parentheses. For example:
    def functionName(parameters):
    Any input parameter or arguments should be placed inside the parenthesis and remember you can have more than one parameter inside.
  • The first statement of a function can also be an optional statement. This is useful since you can describe in string format what the function will be used for or add any other useful information.
  • Remember not to forget the colon (:) at the end as the code block within every function starts with a colon.
  • The statement return exits a function and literally “returns” the value you have decided back to the function. A return statement with no arguments is the same as return None.

A typical function body looks like the code below:

def printThisMessage(string):
    # This program prints  a passed string into this function
    print string

Calling a Function

So we have been through defining a simple function. However after specifying the parameters and structuring the blocks of code you will need to know how to call the function to make use of it. You can execute it from another function or directly from the Python prompt itself.

For example:

def printThisMessage(string):
    # This program prints  a passed string into this function 
    print string

# Now you can call printThisMessage function
printThisMessage(“Hello this is my first message”)
printThisMessage(“Hello this is my second message”)

Output:

Hello this is my first message
Hello this is my second message

The return statement:

So as explained earlier the return statement exits a function and optionally passing back an expression to the caller. I will be showing you an example on how you can return a value from a function.

For example: 

def sumOfNumbers(number1, number2):
    return number1 + number2

#calling the sumOfNumbers function
sumOfNumbers(10,20)

Output:

30

So as you can observe by returning the total in the sumOfNumbers function when the code executed with those parameters the correct total is calculated.
